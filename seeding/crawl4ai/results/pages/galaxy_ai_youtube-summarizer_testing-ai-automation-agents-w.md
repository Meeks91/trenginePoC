# Testing AI Automation Agents: Which One Comes Out on Top?
By Albert HarmonPublished May 16, 2025[Technology](https://galaxy.ai/youtube-summarizer/category/technology)
In a comprehensive test of various AI automation agents including Deep Agent, GenSpark, and others, Deep Agent emerged as the best performer across multiple tasks such as web scraping, research report generation, and app development, while others like Suna and Convergence struggled significantly.
In this blog post, we will explore the results of testing various AI automation agents to determine which one excels in performing a range of tasks. The agents tested include Deep Agent, GenSpark, Manis, Convergence, and Suna. Our goal is to evaluate their performance in tasks such as web scraping, building research reports, and developing applications, while also considering their pricing and features.
## Task 1: Web Scraping
The first task involved scraping data from multiple websites to gather information about funds and ETFs. Each agent was given the same prompt to extract specific data from two different sites. The agents tested were:
  * Deep Agent
  * GenSpark
  * Manis
  * Convergence
  * Suna


### Performance Overview
  * **Deep Agent** : This agent stood out by asking clarifying questions before proceeding, ensuring it understood the requirements. It successfully scraped over 150 funds from the specified websites, earning it an A grade.
  * **GenSpark** : Although it completed the task quickly, it only retrieved data for a limited number of funds, which was insufficient.
  * **Manis** : Similar to GenSpark, it completed the task but failed to gather all the required information.
  * **Convergence** : This agent struggled significantly and did not perform well in this task.
  * **Suna** : It also failed to deliver the expected results, making it a poor performer in this round.


### Conclusion of Task 1
Deep Agent was the clear winner in the web scraping task, demonstrating its ability to handle complex requests effectively.
## Task 2: Research Report Generation
The second task required each agent to create a comprehensive research report on the top five stocks to invest in today. The prompt specified that the report should include photos, charts, analysis, and a bull and bear case for each stock.
### Performance Overview
  * **Deep Agent** : Once again, it excelled by asking relevant follow-up questions and producing a detailed report that aligned with the user's investment horizon and preferences.
  * **GenSpark** : It performed well, generating a solid report that met the requirements.
  * **Manis** : This agent also did a commendable job, receiving an A grade.
  * **Convergence** : Unfortunately, it failed to deliver a usable report, as it encountered issues that required human intervention.
  * **Suna** : Similar to Convergence, Suna did not complete the task satisfactorily.


### Conclusion of Task 2
Deep Agent, GenSpark, and Manis performed well, while Convergence and Suna fell short, particularly in their inability to complete the task without complications.
## Task 3: Building a Calorie Intake Tracking App
The final task involved creating a calorie intake tracking application using modern front-end technologies. Each agent was given a detailed prompt outlining the specifications for the app.
### Performance Overview
  * **Deep Agent** : It produced a functional app that allowed users to set calorie goals and track meals effectively, showcasing its capabilities.
  * **GenSpark** : While it created a decent app, it lacked a feature for adding meals directly, which was a significant oversight.
  * **Manis** : This agent attempted to deploy the app but ultimately failed to deliver a satisfactory product.
  * **Convergence** : It did not complete the task and instead got bogged down in unnecessary research.
  * **Suna** : Similar to Convergence, it failed to provide a usable app, requiring users to sign up to access the output.


### Conclusion of Task 3
Deep Agent once again proved to be the most effective, while GenSpark showed promise but had limitations. Manis, Convergence, and Suna did not meet expectations in this task.
## Pricing and Features Comparison
When considering the pricing of these AI automation agents, here is a breakdown:
  * **Deep Agent** : Starts at $10 per month, with a pro tier at $20 per month.
  * **GenSpark** : Free plan available, with paid plans starting at $24.99 per month.
  * **Manis** : Offers 300 free credits daily, with a subscription of $39 per month.
  * **Convergence** : Priced at $20 per month.
  * **Suna** : Also has a free plan, but it is limited in functionality.


### Additional Features
  * **GenSpark** : Offers additional features like AI slides and AI sheets, making it versatile for various tasks.
  * **Deep Agent** : Provides access to multiple LLMs and custom AI agent creation.
  * **Manis and Suna** : Primarily focused on chat functionalities with limited additional features.
  * **Convergence** : Lacks significant features and has shown a decline in performance.


## Final Thoughts
Based on the tests conducted, Deep Agent emerged as the best AI automation agent, excelling in all tasks and demonstrating a strong understanding of user requirements. GenSpark and Manis also performed well but had limitations. Convergence and Suna struggled significantly, making them less desirable options. For anyone looking to utilize AI automation agents, Deep Agent is highly recommended, while caution should be exercised with the others.
