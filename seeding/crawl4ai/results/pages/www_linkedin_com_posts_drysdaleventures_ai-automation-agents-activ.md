#  Drysdale Ventures’ Post
3,335 followers 
[Twin](https://fr.linkedin.com/company/twinso?trk=public_post-text) raises a $10M seed round to launch the first AI company builder. 🧠🤖 AI is already writing code but most businesses run on operations: sales, support, finance, logistics. Twin lets anyone launch autonomous agents to run these workflows, no code needed. In 1 month of beta, users launched 100,000+ agents. Some ran full businesses. Twin agents handle integrations, fix errors, remember context, and complete 150+ tasks for under $1 ; all in the cloud. ⚡️ A huge round of applause to [Hugo Mercier](https://fr.linkedin.com/in/hugomercier/en?trk=public_post-text), [Joao Justi](https://fr.linkedin.com/in/joao-a-justi?trk=public_post-text), and the founding team for what’s been accomplished so far. 🙌 We’re thrilled to be part of this journey alongside [LocalGlobe](https://uk.linkedin.com/company/localglobe?trk=public_post-text), who leads the round, [Kima Ventures](https://fr.linkedin.com/company/kima-ventures?trk=public_post-text), [Irregular Expressions](https://www.linkedin.com/company/irregular-expressions?trk=public_post-text), [Andrena Ventures](https://www.linkedin.com/company/andrena-ventures?trk=public_post-text) and historical investors like [betaworks](https://www.linkedin.com/company/betaworks?trk=public_post-text) and [Motier Ventures](https://fr.linkedin.com/company/motierventures?trk=public_post-text). 👊 [#AI](https://www.linkedin.com/signup/cold-join?session_redirect=https%3A%2F%2Fwww.linkedin.com%2Ffeed%2Fhashtag%2Fai&trk=public_post-text) [#Automation](https://www.linkedin.com/signup/cold-join?session_redirect=https%3A%2F%2Fwww.linkedin.com%2Ffeed%2Fhashtag%2Fautomation&trk=public_post-text) [#Agents](https://www.linkedin.com/signup/cold-join?session_redirect=https%3A%2F%2Fwww.linkedin.com%2Ffeed%2Fhashtag%2Fagents&trk=public_post-text)
