###  [Week of January 26](https://www.digitalocean.com/blog/whats-new-on-gradient-ai-platform#week-of-january-26)
Product Update [**Now Available: Prompt caching for these Anthropic models**](https://docs.digitalocean.com/products/gradient-ai-platform/details/features/#prompt-caching)
Use prompt caching with [serverless inference chat completion](https://docs.digitalocean.com/products/gradient-ai-platform/how-to/use-serverless-inference/#v1chatcompletions) to cache context and use it in future requests. When you send a request, if part of the input is already cached, you are charged a lower price for those cached tokens, and at the [standard price](https://docs.digitalocean.com/products/gradient-ai-platform/details/pricing/#foundation-model-usage) for the remaining input tokens. This significantly reduces the cost for inference.
We support prompt caching for the following Anthropic models:
  * Claude Sonnet 4
  * Claude Opus 4
  * Claude Opus 4.1
  * Claude Sonnet 4.5
  * Claude Opus 4.5
